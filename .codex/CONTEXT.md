# Context

## Domain Terms

- Loom: voice-first conversational learning environment.
- Mother scene: the core demo session where Loom teaches projectile motion and generates an interactive simulator.
- Artifact: an interactive comprehension aid selected for the subject being learned.
- Visual artifact: a manipulative simulation for quantitative or spatial concepts.
- Connection map: a relational artifact for discursive or conceptual topics.
- Voice-first: spoken interaction is primary; text transcript is supporting context.
- Stable mock: deterministic non-API flow that remains runnable as demo fallback.
- On-the-fly artifact generation: the visible moment where Loom understands the concept, chooses the artifact form, and generates the controlled simulator or map.
- Didactic simulator: a simulator designed for clarity and manipulation rather than exhaustive realism.
